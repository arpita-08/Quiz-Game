# Quiz-Game
An interactive and responsive quiz application built using HTML, CSS, and JavaScript. This project allows users to test their knowledge through a series of questions, track their score in real-time, and view results at the end of the quiz.

Features:-
Start screen with a clean UI
Dynamic question and answer rendering
Real-time score tracking
Progress bar for quiz completion
Result screen with final score display
Restart functionality for replay

Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)

This project demonstrates core front-end development concepts such as DOM manipulation, event handling, and dynamic content updates. It is beginner-friendly and ideal for practicing JavaScript logic and UI design.
